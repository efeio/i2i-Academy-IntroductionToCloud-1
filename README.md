# i2i Academy - Introduction to Cloud Homework

Bu projede aşağıdaki adımlar gerçekleştirilmiştir:
- Google Cloud Platform (GCP) üzerinde Ubuntu 22.04 LTS sanal makinesi (VM) oluşturuldu.
- Yerel bilgisayar terminalinden VM'in genel IP adresine ping testi uygulandı.
- VM'e SSH protokolü ile bağlanılarak `hello.txt` dosyası oluşturuldu ve içeriği doğrulandı.
